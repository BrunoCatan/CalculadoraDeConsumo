## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

-   Vite
-   React
-   Module.css
-   Typescript

## 💻 Projeto

Projeto desenvolvido para facilitar a vida de quem não sabe como calcular o consumo de combustível do seu veiculo.
